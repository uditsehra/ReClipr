This is a clipboard extension application for macOS to make it easier to access clipboard from top menu.

Current features:
    - Basic Clipboard
    - Duplicates Tracking
    - Source Tracking
    - Persistence
    - support for ignoring apps containing sensitive info

This is just a hands on project which I'm doing to get an idea about how things work in macOS as I'm a beginner to mac development. I'm mostly using AI and vibe coding ahead to solve a personal problem.
